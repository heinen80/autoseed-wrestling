# Claude Code Instructions

## Python Files

- Always maintain proper Python indentation when editing any .py files.
- Never use bash heredoc to append to Python files.
- Always edit Python files directly with correct indentation.
- Never introduce smart or curly quotes — always use straight ASCII quotes in all files.
- Always run `python -m py_compile` to verify Python syntax before committing any .py file.
- Never commit code that fails a syntax check.
